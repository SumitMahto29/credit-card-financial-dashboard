# credit-card-financial-dashboard

OVERVIEW

This project is an interactive Power BI dashboard that analyzes credit card transaction data to provide insights into customer spending behavior, revenue generation, and transaction patterns.

The dashboard helps stakeholders monitor:
Total revenue, transaction amount, and interest earned
Revenue trends across quarters
Spending by expense category
Card usage methods (Swipe, Chip, Online)
Revenue contribution by education level, occupation, and card category 



FEATURES
Quarterly Performance Analysis
Track revenue and transaction amounts across Q1–Q4.
Expense Category Breakdown
Identify top spending categories such as Bills, Entertainment, Fuel, Grocery, and Travel.

Card Usage Insights
Compare transaction revenue by Swipe, Chip, and Online payment methods.
Customer Segmentation
Analyze revenue by:
Education level
Occupation
Card category (Blue, Silver, Gold, Platinum)
Interactive Filters

Slice data by:
QuarteR
Gender
Card type
Income level
Week start date



TECHNOLOGIES USED
Power BI – Data visualization and dashboard development
DAX – Measures and calculated fields
Excel / CSV – Source data preparation


Insights from the Dashboard
Bills is the highest revenue-generating expense category.
Swipe transactions dominate revenue compared to Chip and Online methods.
Blue cards contribute the largest share of revenue.
Customers with Graduate education generate the highest revenue segment.
